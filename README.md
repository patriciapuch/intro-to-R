The R Markdown file intro\_r.Rmd contains the notes for the R Ladies Vienna Workshop "Introduction to R" from 29th of October 2025.
The R script "intro\_r.R" contains the code that was presented and the data used for demonstration is located in the folder "iris". The workshop is based on the material of the workshop held by Pia Pfeiffer in October 2024 (see https://github.com/piapfeiffer/intro-to-R).

The R script for part 2 is contained in "intro\_r2.R" and the code adapted live during the workshop is contained in "intro\_r2\_adapted.R".
